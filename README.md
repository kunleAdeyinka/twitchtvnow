# twitchtvnow - https://codepen.io/aadeyinka007/full/ZejqmP


## Built With
* [Jquery](https://jquery.com/) - The frontend scripting language used
* [Bootstrap](http://getbootstrap.com/) - The UI framework used
* [TwitchTV API](https://wind-bow.gomix.me/twitch-api) - The API


## Requirements
A user can perform the following on the application:
*   A user can see whether Free Code Camp is currently streaming on Twitch.tv.
*	A user can click the status output and be sent directly to the Free Code Camp's Twitch.tv channel.
*	If a Twitch user is currently streaming, user's can see additional details about what they are streaming.
*	User will see a placeholder notification if a streamer has closed their Twitch account (or the account never existed).
